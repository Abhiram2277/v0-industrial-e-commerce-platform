## 🔐 Step 1: Verify Domain Ownership in Google Search Console

### Option A: Google Tag Manager (Recommended)
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Click **"Add Property"** → Enter your URL: `https://v0-industrial-ecommerce.vercel.app`
3. Select **Verification Method**: Choose **Google Analytics** or **Google Tag Manager**
4. If you already have Google Analytics connected, verification happens automatically

### Option B: HTML File
1. Download the verification HTML file from Google Search Console
2. Place it in `/public/` directory of your project
3. Redeploy to Vercel
4. Click "Verify" in Google Search Console

### Option C: Meta Tag
1. Copy the meta tag provided by Google Search Console
2. Add to `/app/layout.tsx` in the `<head>`:
```tsx
<meta name="google-site-verification" content="YOUR_VERIFICATION_CODE_HERE" />
```
3. Redeploy to Vercel
4. Click "Verify" in Google Search Console

### Option D: DNS Record (Advanced)
1. Get the DNS record from Google Search Console
2. Add to your domain's DNS settings at your registrar
3. Allow time for DNS propagation
4. Click "Verify" in Google Search Console

---

## 📤 Step 2: Submit Sitemap

1. In Google Search Console, go to **Sitemaps** (left menu)
2. Click **"Add/Test Sitemap"**
3. Enter: `sitemap.xml`
4. Click **"Submit"**
5. Google will crawl your sitemap within 24-48 hours

## 🚀 Step 3: Monitor and Optimize

### Key Metrics to Track:
- **Coverage**: Total pages indexed
- **Performance**: Clicks, impressions, CTR
- **Crawl Stats**: Errors, warnings
- **URL Inspection**: Check if individual pages are indexed

### Common Issues & Solutions:

| Issue | Solution |
|-------|----------|
| Pages not indexed | Check robots.txt, ensure public URLs, verify no noindex tags |
| Crawl errors | Check for 404s, server errors; Fix in code and resubmit |
| Duplicate content | Ensure canonical tags, distinct product descriptions |
| Mobile issues | Test with Mobile-Friendly Test tool in GSC |

## 📱 Additional Optimization Recommendations

### 1. **Add Structured Data (Schema.org)**
Add JSON-LD schema to product pages for rich snippets:
```tsx
<script type="application/ld+json">
{
  "@context": "https://schema.org/",
  "@type": "Product",
  "name": "Product Name",
  "description": "Product description",
  "url": "https://...",
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.5"
  }
}
</script>
```

### 2. **Set Preferred Domain**
In Google Search Console → Settings → Set preferred domain (www or non-www)

### 3. **Enable Enhanced Search Results**
Submit your business info:
- Local Business schema
- Organization schema
- Brand information

### 4. **Monitor Core Web Vitals**
- Check CrUX Dashboard in GSC
- Improve LCP, FID, CLS scores
- Use Google PageSpeed Insights tool
